Repositório do Projeto da Sprint 5

# Dashboard de Análise de Vendas de Carros Usados
Fiz um dashboard que realiza análise de vendas de carros usados nos Estados Unidos, através do arquivos vehices.csv.

# Sobre o Projeto
Criei uma interface de simples visualização que explora e analisa anúncios de carros, através de gráficos e histogramas.

# Funcionalidades
- Analisei a relação entre quilometragem dos veículos através de gráfico de dispersão e histograma;
- Analisei os dados de anúncios de vendas de carros, através de gráficos de dispersão e histogramas, e 
- Viisualizei os dados através de botões e caixa de seleção;

# Tecnologias Utilizadas
- Python;
- Streamlit;
- Plotly Express;
- Pandas.

# Dados Utilizados
O conjunto de dados contém informações sobre anúncios de carros usados, incluindo:
- Preço;
- Quilometragem;
- Ano do modelo;
- Marca e modelo;
- Condição do veículo.
